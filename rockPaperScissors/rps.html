#Rules

  • r = rock ; p = paper ; s = scissors
  • Rock > Scissors
  • Scissors > Paper
  • Paper > Rock
  • Player with greater object wins
  • Use keyboard to Play the game
  • Pressing any other key will leave the user choice to previous selection
    KeyValue
    rRock
    pPaper
    sScissors
  • This is a One(player-1) vs Computer(player-2) Game
  • Refresh after every play to start again
